Return ONE TAB-SEPARATED spreadsheet row in this exact format:

Rules:

BreedWith should default to the opposite sex:
- M cats -> F
- F cats -> M
- ? cats -> ANY

If there is a rainbow icon beside the cat level / age area:
- for M or F cats, set BreedWith to "?"
- for ? cats, keep BreedWith as "ANY"
- do not infer M or F in that case

BreedWith meanings:
- M = can only breed with M cats
- F = can only breed with F cats
- ? = can only breed with ? cats
- ANY = can breed with any gender

Gender should be "M" or "F" for male or female. Some cats are "?" and it should be set to "?"

Stats:
Look at the two numbers for each stat.
Output the RIGHT number exactly as a value from 1-7.

Examples:
- if the right number is 7 -> output 7
- if the right number is 5 -> output 5

The planner logic later treats 7 as complete and anything else as incomplete, but you should still return the actual visible 1-7 value.
